# Azure Certifications

(See [Data Engineer certification](https://learn.microsoft.com/en-us/credentials/browse/?expanded=azure&roles=data-engineer&credential_types=certification) )

Community on reddit: <https://www.reddit.com/r/AzureCertification>

If you are starting from scratch or want to transition into the Microsoft data ecosystem, the standard progression is: 
1. DP-900 Azure Data Fundamental (Foundations) 
2. DP-700 Microsoft Fabric Data Engineer Associate (Core Data Engineering) 
3. DP-600 Microsoft Fabric Analytics Engineer Associate (Data Modeling & Serving). 

## DP-900 Azure Data Fundamental

- <https://learn.microsoft.com/en-us/credentials/certifications/azure-data-fundamentals/?practice-assessment-type=certification>
- Role: Data Engineer (Beginner)
- Course syllabus: <https://learn.microsoft.com/en-us/training/courses/dp-900t00#course-syllabus>

## DP-700 Microsoft Fabric Data Engineer Associate

- <https://learn.microsoft.com/en-us/credentials/certifications/fabric-data-engineer-associate/?practice-assessment-type=certification>
- Role: Data Engineer (Intermediate)

## DP-600 Microsoft Fabric Analytics Engineer Associate

- <https://learn.microsoft.com/en-us/credentials/certifications/fabric-analytics-engineer-associate/?practice-assessment-type=certification>
- Role: Data Engineer, Data Analyst (Intermediate)
