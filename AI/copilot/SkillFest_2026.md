### AI Tools

- `Foundry tools`: cloud-based services --> intergrated with existing applications using prebuilt AI models
- `Microsoft Foundry Portal`: web-based platoform --> build, train & deploy ML models using GUI interface
- `AI Builder`: Microsoft Power Platform --> add AI capabilities to applications
  - Microsoft Power Platform is a comprehensive low-code/no-code development ecosystem. It enables users to build custom business applications, automate workflows, visualize data, and create AI agents without requiring extensive software engineering knowledge.
- `Copilot Studio`: create & manage AI-powered chatbots
- `Semantic Kernel SDK`: open-source SDK, to build agents that can interact with AI models and use responses to enhance your existing code
- `Visual Studio IntelliCode`: extension for Visual Studio, provide AI-powered recommendation
- `ChatGPT`: OpenAI general-purpose conversational AI model
- `GitHub Copilot tools`: a suite of AI-assisted programming tools, including GitHub Copilot (an AI pair programmer). Some Github Copilot features
  - **Agent**: from a request, determines which files to edit, plans the work, and applies changes to your codebase
  - **Plan**: creates a structured, step-by-step implementation plan for a task, which can then be handed off to the Agent for implementation
  - **Ask**: answers questions about your codebase, coding topics, and general technology concepts without making changes to your files
  - **Cloud Agent**: work on code changes for you on the cloud (e.g. GitHub), while letting you decide if and when to open a pull request
  - **Code Review**: AI-generated code review suggestions to help you write better code
