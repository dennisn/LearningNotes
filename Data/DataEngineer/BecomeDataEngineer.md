# Becoming a Data Engineer

- Course URL: <https://app.pluralsight.com/ilx/video-courses/becoming-data-engineer/course-overview>

## Overview

- Data Engineering: building of systems to enable the collection & usage of data
  - Main process: ETL (i.e. Extract - Transform - Load)
  - Stand between external data source & internal data source ==> build/maintain a reliable pipeline & data storage
- Related roles:
  - Data admin: manage the database system --> maintain db performance & integrity
  - Data scientist: use data to extract insights
- Organisation levels of data dependent: data driven, informed, aware, inspired
- Reponsipilities:
  - Weekly: data quality check, visualisation & reporting, and meeting/planning
  - Monthly: Performance report, data security review, technology/legal review, strategic plan
- Core skills:
  - Programming: Python, Java, JavaScript
  - Database
  - Data modeling & warehousing (e.g. data warehouse, data lake)
  - Soft skills: problem solving & communication
- Career path: Data engineer --> Data architect --> Data engineer manager --> Chief Data Officer (CDO)
- Real-life: data engineer may specialize into warehouse, pipeline, cloud, real-time, batch or by technology used (e.g. Snowflake, AWS)

### Data engineer project for learning

  1. Define scope
  2. Find Data source: kaggle, twitter, Coinmarketcap, etc.
     - Use API library to download data (e.g. "github.com/public-apis/public-apis")
  3. Ingestion & automation tool (e.g. github.com/igorbarinov/awesome-data-engineering)
  4. Transformation
  5. Data storage & processing: batch or stream
  6. Reporting & visualization
  7. Documentation
