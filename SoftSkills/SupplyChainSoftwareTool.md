# Emergent trends
  - Three main trend: 
    + Big Data and Real-Time Analytics: let user monitor trends (sales, markets, etc) in real time and quickly respond to demands
    + Cloud-based migration: optimise operation, increase integration while reduce upfront cost, but still have access to latest tools
    + Apply AI: assist in analysis/decision making, using natural language
